1. Pendahuluan

Tentang E-Magazine Sekolah

E-Magazine Sekolah adalah sistem manajemen konten digital yang dirancang khusus untuk lingkungan pendidikan. Sistem ini memungkinkan siswa, guru, dan admin untuk berkolaborasi dalam menciptakan, mengelola, dan mempublikasikan artikel digital.

3. Mengakses Sistem

3.1 Registrasi Akun Baru

1Buka Halaman Registrasi

Akses URL: http://localhost/E-Magazine/register.php

2Isi Form Registrasi

Lengkapi data berikut:

  • Email: Email aktif yang valid
  • Username: Minimal 3 karakter
  • Password: Minimal 8 karakter dengan huruf besar, kecil, dan angka

3Konfirmasi Registrasi

Setelah submit, akun akan pending menunggu aktivasi admin.

Penting: Hubungi admin untuk aktivasi akun dan penentuan role.

3.2 Login ke Sistem

1Buka Halaman Login

Akses URL: http://localhost/E-Magazine/login.php

2Masukkan Kredensial

Isi username dan password yang telah terdaftar.

3Redirect ke Dashboard

4. Penggunaan Area Publik

4.1 Membaca Artikel

1Navigasi Beranda

Halaman beranda menampilkan artikel dalam dua kolom:

  • Kiri: Artikel terbaru (card lengkap)
  • Kanan: Artikel populer (compact)

2Buka Detail Artikel

Klik judul atau gambar artikel untuk membuka halaman detail.

4.2 Like dan Komentar

1Memberikan Like

Klik tombol "Like" dengan ikon hati pada artikel.

2Menulis Komentar

Scroll ke bawah artikel dan isi form komentar.

5. Dashboard Siswa

5.1 Menulis Artikel

1Akses Menu Tambah Artikel

Dari dashboard siswa, klik "Tambah Artikel".

2Isi Form Artikel

Lengkapi form dengan:

  • Judul artikel
  • Kategori
  • Gambar (opsional)
  • Konten artikel

3Upload Gambar

Klik "Choose File" untuk upload gambar artikel (max 5MB).

4Submit Artikel

Klik "Submit Artikel" setelah semua data terisi.

Artikel akan berstatus "pending" menunggu review guru.

5.2 Mengelola Artikel

1Lihat Daftar Artikel

Menu "Artikel Saya" menampilkan semua artikel yang telah ditulis.

2Edit Artikel

Klik "Edit" pada artikel dengan status Draft atau Rejected.

5.3 Mengelola Profil

1Update Profil

Di menu "Profil", ubah nama, email, atau password.

2Upload Foto Profil

Pilih avatar default atau upload foto pribadi.

6. Dashboard Guru

6.1 Moderasi Artikel

1Akses Menu Moderasi

Dari dashboard guru, klik "Moderasi".

2Review Artikel

Baca artikel siswa yang pending dan berikan keputusan.

3Berikan Keputusan

Pilih aksi: Approve, Reject, atau Request Revision.

6.2 Publikasi Artikel

1Menu Publikasi

Kelola artikel yang sudah disetujui untuk publikasi.

7. Dashboard Admin

7.1 Mengelola User

1Daftar User

Menu "Users" menampilkan semua pengguna sistem.

2Aktivasi Akun

Tentukan role dan aktifkan akun user baru.

8. Tips dan Troubleshooting

Tips Penggunaan

  • Gunakan judul artikel yang menarik dan deskriptif
  • Kompres gambar sebelum upload (max 5MB)
  • Pilih kategori yang tepat untuk artikel
  • Proofread artikel sebelum submit

Masalah Umum

Masalah Solusi
Tidak bisa login Pastikan akun sudah diaktifkan admin
Upload gambar gagal Cek ukuran file (max 5MB) dan format (JPG/PNG)
Artikel tidak muncul Tunggu approval guru atau cek status di "Artikel Saya"

9. FAQ (Frequently Asked Questions)

Q: Bagaimana cara mengubah password?

A: Masuk ke menu "Profil", isi password lama dan baru, lalu klik "Update".

Q: Berapa lama proses review artikel?

A: Biasanya 1-3 hari kerja, tergantung workload guru.

Q: Apakah ada batasan jumlah artikel?

A: Tidak ada batasan. Siswa dianjurkan aktif menulis.

Q: Bisakah mengedit artikel yang sudah published?

A: Siswa tidak bisa. Hubungi guru jika ada koreksi penting.
